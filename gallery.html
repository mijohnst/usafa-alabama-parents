Setup Required

  1. Go to script.google.com, create a new project, and paste the Apps Script code (provided separately).
  2. Click Deploy → New deployment, choose Web app, set Execute as: Me and Who has access: Anyone. Copy the web app URL.
  3. In gallery.html, replace YOUR_APPS_SCRIPT_URL with that URL.
  4. Share your "Club Photos" Drive folder as Anyone with the link → Viewer.
