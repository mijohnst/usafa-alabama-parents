# USAFA Parents Club of Alabama

Official website for the United States Air Force Academy Parents Club serving the state of Alabama.

## About

This website serves as the online hub for Alabama families with cadets attending or appointed to the U.S. Air Force Academy. The club provides:

- Support and community for cadet families
- Annual scholarship programs
- BCT care package coordination
- New family orientation events
- Parent mentorship programs
- Liaison coordination throughout Alabama

## Website Features

- Responsive design for mobile and desktop
- Event calendar
- Leadership roster
- Scholarship information
- Contact form
- Programs and support information

## Deployment

This site is deployed via GitHub Pages and is accessible at the repository's GitHub Pages URL.
